# Dead Reckoning System
 


| Task     | Execution time   | Relative Deadline  | Periodicity  | Priority
| ------------- |:----------:| -----:| -----:|-----:|
| GPS Task | 1ms | 100ms | 100ms |2|
| CAN Task | 5ms | 100ms | 100ms |1|

*all times are in ticks set at 1kHz, therefore 1000 ticks = 1 second*
